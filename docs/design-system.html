03 · Colour

The 7 Horizon colour schemes

Defined in config/settings_data.json and selectable per-section in the theme editor. Each card is rendered with that scheme's exact values: heading, body text, link/primary colour, both button styles, an input, and variant pickers (idle + selected). Hover the buttons/inputs to see the configured hover states. Note: the client can edit these in the theme editor, so always shopify theme pull before trusting local values.

04 · Type

Typefaces

Both faces are self-hosted in assets/ (OFL licence) because Shopify's font picker doesn't carry them.
